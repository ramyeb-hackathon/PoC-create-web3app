## create-web3app script

### Steps : 

1 - Clone the repo


2 - Install dependencies
```
pnpm install --prefix ./PoC-create-web3app 
```

3 - Create a new directory

```
mkdir app 
cd ./app 
```

3 - Init folder & Link the project

```
pnpm init
pnpm link ../PoC-create-web3app
```

4 - Launch the script

```
pnpm create-web3app
```
